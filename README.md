# china-wanmeisport-lab

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖不同主题与功能场景。仓库本身不针对任何特定域名或网站，仅作为静态页面的集合与存档。

## 项目简介

`china-wanmeisport-lab` 是一个以 HTML 页面为主的归档仓库。每个页面均为独立自包含的 HTML 文件，可直接在浏览器中打开使用。仓库旨在提供一种轻量、灵活的方式，用于保存和分享零散的页面内容。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
├── index.html      # 仓库主页索引（可选）
└── README.md       # 本文件
```

- `pages/`：核心目录，每个子文件夹或文件对应一个独立页面。
- `assets/`：存放页面可能引用的公共资源，便于复用与维护。
- `index.html`：可选的入口页面，用于列出所有归档页面链接。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间相互独立，无耦合依赖。
- 部分页面可能引用 `assets/` 下的公共资源，建议在本地或托管环境中保持目录结构完整。
- 归档文件命名尽量体现内容主题，但不包含具体域名或营销信息。

## 维护说明

- 欢迎提交 Pull Request 增加新的独立 HTML 页面，或对现有页面进行改进。
- 新增页面请放入 `pages/` 目录，并保持命名清晰。
- 如引用外部资源，请尽量使用相对路径或注明来源。
- 仓库不保证所有页面长期可用或兼容所有浏览器，但会尽力保持基本可用性。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件。
3. 也可将整个仓库部署到任意静态托管服务（如 GitHub Pages），通过网址访问。
